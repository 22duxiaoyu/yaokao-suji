# 两分钟面试演示脚本

## 0:00 - 0:20 产品问题

“这是我做的执业药师 AI 速记工具。用户不用手工制卡，上传自己的讲义或错题后，系统会拆成可追溯的知识片段，再生成一张只考一个点的卡片。”

展示资料首页和个人知识架构。

## 0:20 - 0:55 AI 出卡

粘贴一段准备好的药理资料，点击导入和拆解。

讲解：

- 文件先解析分块
- 检索结果保留来源位置
- DeepSeek 使用 JSON 结构化输出
- 模型失败时有确定性兜底，不会让流程中断

打开一张生成卡，指出原文来源、人工修改和批量确认。

## 0:55 - 1:25 复习闭环

进入复习页：

- “记住、模糊、忘记”控制记忆排期
- “标记”表示内容不准确，不会当成薄弱知识继续背
- “收藏”表示主观重点

点击“忘记”，展示薄弱卡 AI 辅助记忆页面。

## 1:25 - 1:50 工程设计

打开 README 架构图：

- Next.js 全栈
- PostgreSQL + Drizzle
- 本地 / Supabase 双文件存储
- AI 调用日志与每日限额
- RAG 检索日志和来源追溯
- Android WebView APK

说明作品集阶段使用单 Demo 用户，但数据模型预留了正式多租户演进路径。

## 1:50 - 2:00 结尾

“这个项目不只是调一个模型 API。我重点解决了资料如何变成可信卡片、模型失败怎么办、用户反馈怎样进入下一轮复习，以及如何用接近零成本交付一个可安装的完整 Demo。”

阿司匹林通过不可逆抑制环氧化酶，减少血栓素 A2 生成，从而抑制血小板聚集。主要不良反应包括胃肠道刺激和出血，活动性消化性溃疡患者应避免使用。儿童病毒感染时使用阿司匹林需要警惕瑞氏综合征。
